1、Doctype是什么？如何触发严格模式和混杂模式？区分它们有何意义？

Doctype是document type的缩写，用来说明所用的XHTML和HTML的版本。

XHTML提供了三种DTD声明，分别是:

类型 完整声明
过渡的 <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-tranisitional.dtd">
严格的 <DOCTYPE html PUBLIC "-W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
框架的 <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

2、块级元素和行内元素各有那些？

行内元素，会在同一行内。

块级元素会各占一行。

行内元素主要有：sapn、a、b、img、br、button、strong、textarea、select

块级元素主要有：div、ul、li、p、fieldset、form、h1-h6、hr、table、tr、td

4、导入CSS的方式

  1. 行内试：行内式是在标记的style属性中设定CSS样式。
  2. 嵌入式：嵌入式是将CSS样式集中写在网页的<head></head>标签对的<style></style>>标签对中。
  3. 链接式：例如：@import"mystyle.css"; 此处要注意.css文件的路径。
  4. 导入式：例如：<link href="mystyle.css" rel="stylesheet" type="text/css"/>

5、CSS选择符有哪些

CSS选择符主要有一下十种：

  1. 通配选择符：*{margin:0px}
  2. 类型选择符：a{text-decoration:none;}
  3. 属性选择符：input[type="button"]{padding:5px}
  4. 包含选择符：div .code a{text-decoration:none}
  5. 子对象选择符：ul .test>li{font-size:14px}
  6. ID选择符：#test{color:red}
  7. 类选择符：.test{color:red}
  8. 选择符分组：body,ul,li{margin:0px;}
  9. 伪类及伪对象选择符：div:first-letter{font-size:16px;}
  10. 相邻选择符：li+li{font-weight:bold;}

6、前端主要分为哪三层，分别是什么？作用是什么？

  1. 结构层：由HTML或XHTML之类的标记语言负责创建，仅负责语义的表达。解决了“内容是什么”的问题。
  2. 表现层：由CSS负责创建，解决了页面“如何显示内容”的问题
  3. 行为层：由脚本决定。解决了页面上“内容应该如何对事件做出反应”的问题

9、经常遇到的浏览器兼容问题有哪些？如何解决？

  1. 浏览器默认的margin和padding不同。解决方案是加一个全局的*{margin:0;padding:0;}来统一。
  2. IE6双边距bug:块属性标签float后，又有横行的margin情况下，在ie6显示margin比设置的大。解决方案是在float的标签样式控制中加入 display:inline;将其转化为行内属性。
  3. 在ie6，ie7中元素高度超出自己设置高度。原因是IE8以前的浏览器中会给元素设置默认的行高的高度导致的。解决方案是加上overflow:hidden或设置line-height为更小的高度。
  4. min-height在IE6下不起作用。解决方案是添加 height:auto !important;height:xxpx;其中xx就是min-height设置的值。
  5. 透明性IE用filter:Alpha(Opacity=60)，而其他主流浏览器用 opacity:0.6;
  6. a(有href属性)标签嵌套下的img标签，在IE下会带有边框。解决办法是加上a img{border:none;}样式。
  7. 父子标签间用margin的问题，表现在有时除IE(6/7)外的浏览器子标签margin转移到了父标签上，IE6&7下没有转移。解决办法就是父子标签间的间隔建议用padding，兄弟标签间用margin。
  8. Opera：Presto内核

10、如何居中一个浮动元素？

父元素和子元素同时左浮动，然后父元素相对左移动50%，再然后子元素相对右移动50%，或者子元素相对左移动-50%也就可以了。

11、如何管理CSS文件、JS图片？

  1. 对各个项目中CSS，JS里的稳定的通用代码进行提取，形成公共文件，然后利用CDN等资源进行缓存，减轻服务器压力。
  2. 去掉JS、CSS里的冗余代码，对代码进行精减。
  3. 对JS、CSS进行压缩合并，减少请求次数。
  4. 对页面上的小图标，背景等图片进行合并，减少请求次数。
  5. JS、CSS、图片均用版本控制工具进行管理，方便修改与恢复。

13、xhtml和html有什么区别

HTML是一种基本的WEB网页设计语言，XHTML是一个基于XML的置标语言
最主要的不同：
XHTML 元素必须被正确地嵌套。
XHTML 元素必须被关闭。
标签名必须用小写字母。
XHTML 文档必须拥有根元素。
